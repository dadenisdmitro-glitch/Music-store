# Music-store

Короткое руководство и заметки по дизайну для проекта `Music-store`.

**Что добавлено**:
- `style.css` — адаптивная тема с сеткой для карточек товаров и модальным окном корзины.
- `script.js` — простой рендеринг демонстрационных товаров и базовая логика корзины.
- Обновлён `index.html` — добавлены кнопка корзины и контейнер для динамических карточек.

Открыть сайт локально: просто запустите статический сервер или откройте `index.html` в браузере.

Пример команды (в каталоге проекта):
```
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

Дальше можно:
- Заменить файлы-плейсхолдеры в `images/` на реальные обложки альбомов.
- Добавить страницу товара и корзину на бэкенд для полноценного оформления заказа.
- Подключить библиотеку сетки или компонентную систему при необходимости.

Если хотите, могу:
- добавить адаптивное меню для мобильных устройств;
- сделать страницу отдельного товара и переходы;
- связать корзину с локальным хранилищем (`localStorage`) или API.
